The Making of Scenic Music

Once the user inputs an address or a general location, we used google's Geocoder API in order to get the coordinates of that location. Those coordinates are then translated through tripadvisor's API into a unique location id. From there, we get information about the place, and using Spotify's API we are able to display relevant and appropriate music to the scenery, hence our name Scenic Music.

Feel free to check out our code on Github!
