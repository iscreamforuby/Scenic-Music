The making of Scenic Music

Once the user inputs an address or a general location, we used google's Geocoder api in order to get the coordinates (latitude and longitude) of that location. Then, we plugged those coordinates into tripadvisor's api to retrieve the location id. From that we're able to see the description of the place searched up. Using that we inserted it into the Spotify's api in order to generate music relevant and appropriate to the scenery, hence our name Scenic Music.
